一、水平居中对齐（共 7 种）

parent1 行内: child1

1、行内元素：text-align: center（文本居中）

        
          .parent1 {
            text-align: center;
          }
parent2
块级(定宽): child2

2、块级元素 margin: 0 auto

          
          .child2 {                   
            width: 160px;                   
            margin: 0 auto;                   
          }
parent3
块级(定宽): child3

3、块级元素：flex 一把梭（IE8不支持）

          
    .parent3 {
      display: flex;
      justify-content: center;
    }
          
          
    .child3 {
      width: 130px;
      height: 50px;
    }
parent4
块级(定宽): child4

4、块级元素：绝对定位+transform（元素的宽度未知，高度未知。IE8不支持） 

          
    .parent4 {
      position: relative;
    }
          
          
    .child4 {
      position: absolute;
      left: 50%;
      transform: translateX(-50%);
      width: 130px;
    }
parent5
块级(定宽): child5

5、块级元素：绝对定位+margin-left（元素高度和宽度已知时）

          
    .parent5 {
      position: relative;
    }
          
          
    .child5 {
      position: absolute;
      left: 50%;
      margin-left: -65px;
      width: 130px;
    }
parent6
块级(定宽): child6

6、块级元素：绝对定位+left/right:0;+margin

            
      .parent6 {
        position: relative;
      }
            
            
      .child6 {
        position: absolute;
        left: 0;
        right: 0;
        margin: 0 auto;
        width: 130px;
      }
parent7
块级(float): child7

7、块级元素（float）：fit-content + margin:auto

            
      .parent7 {
        margin: auto;
        width: fit-content;
      }
            
            
      .child7 {
        float: left;
      }

二、垂直居中对齐（共 6 种）

parent1 行内: child1

1、行内元素：line-height（文本居中）

          
      .parent1 {
        height: 210px;
      }
          
          
            .child1 {
              line-height: 210px;
            }
parent2
行内块元素: child2

2、行内块元素：伪元素+vertical-align（IE7适用）

                
  .parent2::after,
  .child2 {
    display: inline-block;
    vertical-align: middle;
  }
                
                
          .parent2::after {
            content: '';
            height: 100%;
          }
parent4
块级：child4

4、Flex 弹性盒子（IE8/IE9不支持）

            
          .parent4 {
            display: flex;
            align-items: center;
          }
parent5
块级：child5

5、绝对定位 + transform（元素的宽度未知，高度未知。IE8不支持。） 

      
      .parent5 {
        position: relative;
      }
      
      
    .child5 {
      position: absolute;
      top: 50%;
      transform: translate(0, -50%);
    }
parent6
块级：child6

6、绝对定位 + margin-top（元素高度和宽度已知时，适用所有浏览器） 

          
      .parent1-6 {
        position: relative;
      }
          
          
      .child1-6 {
        position: absolute;
        top: 50%;
        height: 50px;
        margin-top: -25px;
      }
parent7
块级：child7

7、绝对定位+top/bottom: 0;（注意子绝父相；子盒子不给高度会充满父盒子） 

            
        .parent7 {
          position: relative;
        }
            
            
        .child7 {
          position: absolute;
          top: 0;
          bottom: 0;
          margin: auto 0;
          height: 50px;
        }

三、综合垂直水平居中（常用 7 种）

parent1 行内: child1

1、行内元素：text-align: center（文本居中）

        
        .parent1 {
          text-align: center;
          height: 220px;
        }
        
        
        .child1 {
          line-height: 220px;
        }
parent2
块级: child2

2、块级元素：flex（对于行内、行内块均适用）

          
          .parent2 {
            display: flex;
            justify-content: center;
            align-items: center;
          }
parent3 块级: child3

3、绝对定位 + transform（适用于未知宽高，行内、行内块；不适用 IE8） 

          
    .parent3 {
      position: relative;
    }
          
          
  .child3 {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
  }
parent4
child4

4、绝对定位 + margin 负边距（适用于高宽已知，适用所有浏览器）

          
    .parent4 {
      position: relative;
    }
          
          
  .child4 {
    position: absolute;
    height: 50px; width: 100px; 
    top: 50%; left: 50%;
    /* margin 为负数，且为高宽的一半 */
    margin: -25px -50px;
  }
parent5
child5

5、grid 单个元素垂直居中

          
    .parent5 {
      display: grid;
      /* 或者 place-items: center; */
    }
          
          
        .child5 {
          margin: auto;
        }
parent6
块级(定宽高): child6

6、定宽高块级元素：left/right/top/bottom 均设为0，margin设为 auto 

            
      .parent6 {
        position: relative;
      }
            
             
      .child6 {
        position: absolute;
        left: 0; right: 0;
        top: 0; bottom: 0;
        margin: auto;
        width: 130px; height: 50px;
      }
parent7
块级child7

7、table-cell 强制表格单元化（父子元素均定宽高；不常用，某些情况有问题） 

            
      .parent2-7 {
        display: table-cell;
        vertical-align: middle;
        height: 200px;
        width: 200px;
      }
            
            
      .child2-7 {
        margin: auto;
        height: 50px;
        width: 50px;
      }
